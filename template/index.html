    Cheat Sheet – Shortcuts

    Key Function
    enter/space/right arrow advance
    backspace/left arrow go back
    g prompt: go to slide
    home/pos1 jump to start slide
    end jump to end slide
    shift +/- zoom in/out
    F11 fullscreen mode
